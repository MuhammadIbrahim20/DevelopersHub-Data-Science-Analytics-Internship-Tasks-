💳 Task 2: Credit Risk Prediction
The Task Objective: Develop a robust binary classification model to predict whether a loan applicant is likely to default based on their financial and demographic background.

 Performed missing value handling using mode imputation for categorical attributes and median imputation for continuous data points. Executed Exploratory Data Analysis (EDA) on key fields like loan amounts, education levels, and incomes. Trained Logistic Regression and Decision Tree models, evaluating performance via confusion matrices and accuracy metrics.

Results and Insights: An applicant's historical credit footprint (Credit_History) is the single most critical feature predicting loan approval viability. Applicants with an existing positive credit record display a dramatically higher success rate.
