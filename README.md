# Data Science & Analytics Internship Portfolio

## 📌 Project Overview
This repository contains the completed deliverables for the Data Science & Analytics Internship at **DevelopersHub Corporation**. The objective of these tasks is to build core competencies in exploratory data analysis (EDA), statistical visualization, data preparation, predictive machine learning modeling, and rigorous performance evaluation using Python.

All analysis, cleaning pipelines, models, and outcome summaries are organized into dedicated Jupyter Notebooks.

---

## 🛠️ Tech Stack & Ecosystem
- **Environment:** Jupyter Notebook
- **Data Wrangling:** Pandas, NumPy
- **Data Visualization:** Matplotlib, Seaborn
- **Machine Learning Architecture:** Scikit-Learn
