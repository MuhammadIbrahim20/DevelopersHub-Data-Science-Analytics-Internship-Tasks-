📊 Task 1: Exploring and Visualizing a Simple Dataset
The Task Objective: Understand how to properly read, systematically summarize, and profile a structural dataset using statistical visualization tools.

Your Approach: Loaded the dataset using Pandas and profiled its layout via .shape, .columns, and .head(). Used Seaborn and Matplotlib to construct multi-variable scatter plots, histograms to evaluate feature density spreads, and box plots to isolate outliers.

Results and Insights: The Iris-setosa species is completely linearly separable from both Iris-versicolor and Iris-virginica, particularly when evaluating the geometric boundary between petal length and petal width.
